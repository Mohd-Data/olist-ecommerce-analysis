This project focuses on transforming and analyzing an e-commerce dataset (Olist) using modern data tools. Raw data
was extracted from Kaggle and processed using Python for data cleaning and preprocessing, ensuring analysis-ready
datasets. The cleaned data was ingested into Google BigQuery, where scalable datasets were designed for efficient
querying of large relational data. Data transformation was performed using dbt, implementing staging, intermediate,
and mart layers to build an analytics-ready data model. Exploratory data analysis (EDA) was conducted using SQL
to derive insights on revenue trends, customer behavior, delivery performance, and product category performance.
Finally, BigQuery was integrated with Power BI to build data models, create DAX measures, and develop interactive
dashboards visualizing key KPIs such as revenue, order trends, and customer segmentation.
